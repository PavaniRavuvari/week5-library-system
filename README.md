# Library Management System

## Project Description
A Library Management System built using Python and Object-Oriented Programming principles.  
It allows librarians to manage books, members, and borrowing operations efficiently.

## Features
- Add and manage books
- Register library members
- Borrow and return books
- Due date tracking
- JSON file-based data persistence
- Menu-driven user interface

## Technologies Used
- Python 3
- Object-Oriented Programming
- File Handling (JSON)
- Git & GitHub

## How to Run
```bash
python -m library_system.main
